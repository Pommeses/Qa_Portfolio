| Frage                                                                                        | Antwort                                 |
| -------------------------------------------------------------------------------------------- |-----------------------------------------|
| Können nur registrierte Nutzer Produkte bewerten, oder auch Gäste?                           | Nur registrierte Nutzer                 |
| Ist eine Bewertung erst nach einem Kauf möglich?                                             | Ja                                      |
| Dürfen Nutzer ihre Bewertung nachträglich ändern oder löschen?                               | Nein, wenn kein Kauf getätigt wurde     |
| Wird die Durchschnittsbewertung automatisch angezeigt (z. B. Sterne unter dem Produktnamen)? | Ja, aber nur auf der Produktdetailseite |
| Muss das Alter nur einmalig bestätigt werden oder bei jedem erneuten Seitenaufruf?           | Bei jedem Seitenaufruf                  |
| Wird das Alter gespeichert (z. B. Cookie oder Nutzerprofil)?                                 | Ja                                      |
| Wie wird verhindert, dass Minderjährige den Schutz umgehen (z. B. einfach „Ja“ klicken)?     | Eingabe des Geburtsdatums               |
